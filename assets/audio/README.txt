Audio File Instructions
=======================

Place your background music file here with the name:
"romantic-instrumental.mp3"

Recommended:
- Format: MP3 (best browser compatibility)
- Duration: 2-4 minutes (will loop automatically)
- File size: Under 5MB for faster loading
- Style: Romantic instrumental (piano, strings, acoustic)

Popular choices:
- "Sailor Song" (instrumental version)
- "A Thousand Years" (instrumental)
- "Canon in D" (modern arrangement)
- "River Flows in You"
- Custom romantic instrumental

Alternative formats supported:
- romantic-instrumental.wav
- romantic-instrumental.ogg

Note: If you use a different filename, update line 17 in index.html:
<source src="assets/audio/YOUR-FILENAME.mp3" type="audio/mpeg">

For copyright-free music, check:
- YouTube Audio Library
- Free Music Archive
- Incompetech
- Bensound
